# nanobanana/queue/ui_icons.prompt.md

## Asset Information
- **Filename**: ui_icons.png
- **Size**: 128x128 sprite sheet
- **Purpose**: HUD and Inventory icons
- **Target**: assets/resources/sprites/ui/

## Nano Banana Command
/generate "16-bit pixel art RPG UI icons..." --styles="pixel art" --count=1

## Prompt
16-bit pixel art UI icons for Inventory.
Style: Simple, readable, colorful.

Icons Required (16x16 or 24x24 each):
1. **Heart**: Full, Half, Empty (for HUD)
2. **Sword**: Basic iron sword
3. **Shield**: Basic wooden shield
4. **Bomb**: Black bomb icon
5. **Boomerang**: Wooden boomerang icon
6. **Key**: Small key, Boss key
7. **Rupee**: Green, Blue, Red
8. **Potion**: Red potion bottle

Output:
- Grid layout
- Transparent background

## Acceptance Criteria
- [ ] Includes all requested icons
