# nanobanana/queue/overworld_tileset.prompt.md

## Asset Information
- **Filename**: overworld_tileset.png
- **Size**: 256x256 pixel grid (16x16 tiles)
- **Purpose**: Overworld environment tileset
- **Target**: assets/resources/sprites/tilesets/

## Nano Banana Command
/generate "16-bit pixel art overworld tileset..." --styles="pixel art" --count=1

## Prompt
16-bit pixel art overworld tileset
For a Zelda-like top-down 2D action RPG.

Mood/Theme:
- Hyrule Field / peaceful countryside aesthetic
- Bright, colorful, inviting atmosphere
- Classic SNES-era visual style

Required Tiles (16x16 each):
- Grass: Plain, with flowers, tall grass
- Dirt: Path tiles (straight, corner)
- Water: River/Pond edge tiles
- Nature: Tree (2x2 size usually), Bush
- Cliff: Rock wall tiles (top/bottom/sides)

Output:
- Strictly aligned 16x16 grid
- Organized blocks for autotiling (corners, edges)

## Acceptance Criteria
- [ ] 16x16 grid alignment
- [ ] Includes Grass, Dirt Path, Water, Cliff
- [ ] Includes Tree and Bush assets
