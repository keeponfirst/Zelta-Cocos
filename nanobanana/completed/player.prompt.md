## Prompt
16-bit pixel art RPG character sprite sheet

A young hero character for a Zelda-like top-down 2D action RPG game.

Style:
- 16-bit SNES era pixel art style
- Similar to Link from A Link to the Past
- Limited color palette (12-16 colors)
- Clean pixel art with defined outlines
- Heroic but approachable design

Character Design:
- Young adventurer with simple tunic and shorts
- Green/teal primary color scheme
- Brown belt with item pouch
- Simple short hair
- Determined expression
- No complex accessories in base design

Sprite Requirements:
- 32x32 pixel character sprites
- 4 directional facing (front, back, left, right)
- Idle pose: standing relaxed, slight breathing animation
- Walk cycle: 4 frames per direction
- Attack pose: sword swing arc, 3 frames
- Hurt pose: knocked back, flashing

Background: 
- Transparent (checkerboard pattern for reference)
- No drop shadow on character

Output:
- Sprite sheet layout with clear grid
- Each frame clearly separated
- Include color palette reference strip

## Acceptance Criteria
- [ ] Strictly 32x32 grid alignment
- [ ] 4 directions visible (Front, Back, Left, Right)
- [ ] No fuzzy edges (clean pixel art)
- [ ] Transparent background
