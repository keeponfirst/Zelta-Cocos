# nanobanana/queue/dungeon_tileset.prompt.md

## Asset Information
- **Filename**: dungeon_tileset.png
- **Size**: 256x256 pixel grid (16x16 tiles)
- **Purpose**: Dungeon environment tileset
- **Target**: assets/resources/sprites/tilesets/

## Nano Banana Command
/generate "16-bit pixel art dungeon tileset..." --styles="pixel art" --count=1

## Prompt
16-bit pixel art dungeon tileset
A visual reference board for creating an indoor dungeon tileset for a Zelda-like top-down 2D action RPG.

Mood/Theme:
- Classic Zelda dungeon aesthetic
- Dark, mysterious, slightly dangerous
- Ancient temple/ruin atmosphere
- Stone gray color palette with moss/torch accents

Required Tiles (16x16 each):
- Floor: Stone brick, cracked variant, decorative pattern
- Wall: Solid stone wall (top), side walls
- Hazards: Black pit/void, Spike trap
- Decor: Torch on wall, Pillar base

Output:
- Strictly aligned 16x16 grid
- Organized blocks of related tiles
- Clear separation between wall/floor types

## Acceptance Criteria
- [ ] 16x16 grid alignment
- [ ] Includes Wall (Front/Side) and Floor variants
- [ ] Includes Pit/Void tiles
